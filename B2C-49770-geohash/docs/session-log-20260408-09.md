3. **[High]** axios 재시도 시 이중 암호화 → 서버 복호화 실패
4. **[High]** 아파트 화면만 sendEncryptedLocation try/catch 누락
5. **[High]** sendEncryptedLocation에서 accessToken 없이 POST 전송
6. **[Medium]** interceptor의 빈 catch 블록
7. **[Medium]** 5개 화면 호출 패턴 불일치

### 코드리뷰 반영 (1, 3, 4, 5번 수정)
- axios interceptor `_geohashEncrypted` 플래그 추가 (이중 암호화 방지)
- 5개 맵 화면 `.catch(() => {})` 패턴 통일
- sendEncryptedLocation accessToken 미존재 시 early return
- console.log 출력 제거
