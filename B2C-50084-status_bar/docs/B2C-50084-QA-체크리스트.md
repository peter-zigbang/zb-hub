- [ ] 학교 상세 > 위치보기
- [ ] 단지 상세 > VR 홈투어 (랜선홈투어)
- [ ] 단지 상세 > 부동산 계산기

- [ ] 집 내놓기 > 소개 페이지
- [ ] 방 내놓기 > 주소 검색

## 확인 사항

### 공통
- [ ] 상단 네비게이션 바가 시스템 상태바와 겹치지 않음
- [ ] 상태바 영역 배경색이 불투명(흰색 또는 해당 화면 배경색)으로 표시
- [ ] X(닫기) 또는 Back 버튼 정상 동작
- [ ] 화면 스크롤 정상 동작

### 집 내놓기 > 소개 페이지 (특별 확인)
- [ ] X 버튼 노출 확인
- [ ] WebView 콘텐츠 스크롤 정상 동작
- [ ] 스크롤 시 헤더 배경 opacity 변경 정상 동작

### 상가 > 이미지 선택 (특별 확인)
- [ ] 이미지 카운터/닫기 버튼이 상태바 아래에 정상 위치
